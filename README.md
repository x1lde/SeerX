A cloud-integrated, end-to-end algorithmic trading platform that predicts stock and cryptocurrency price movements using machine learning (`scikit-learn`), stores predictions and transaction history in `Supabase`, and executes automated paper trades through the `Alpaca API`. SeerX provides a fully automated pipeline—from data acquisition and model inference to trade execution and cloud-based monitoring.

## Database Architecture (Supabase / PostgreSQL)

### `predictions`

Stores every prediction generated by the machine learning model.

```sql
create table public.predictions (
  id bigint generated always as identity primary key,
  date date not null,
  symbol text not null,
  predicted_direction text not null,
  created_at timestamptz default now()
);
```

### `trades`

Stores executed paper trades through the Alpaca API.

```sql
create table public.trades (
  id bigint generated always as identity primary key,
  date date not null,
  symbol text not null,
  side text not null,
  qty numeric not null,
  alpaca_order_id text,
  created_at timestamptz default now()
);
```

## Local Installation

```bash
git clone https://github.com/<username>/SeerX.git
cd SeerX

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

### Environment Variables

```env
SUPABASE_URL=
SUPABASE_KEY=

ALPACA_API_KEY=
ALPACA_SECRET_KEY=
```

## Cloud Automation

GitHub Actions automatically executes scheduled workflows for data collection, prediction generation, and paper trading.

Repository secrets required:

- `SUPABASE_URL`
- `SUPABASE_KEY`
- `ALPACA_API_KEY`
- `ALPACA_SECRET_KEY`
